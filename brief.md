# 연구 주제
본 연구는 기업의 정보화 지원 및 투자와 AI의 실제 활용이 프로세스 개선 및 운영 내재화와 어떤 관련을 가지는지 분석하고자 한다. 이를 위해 2019~2024년 NIA 정보화통계조사 데이터를 활용한다.

# 전처리 원칙
## 분석용 데이터셋 구분
- 메인 working data : 2020 ~ 2024
	- 메인 Y, 보조 Y, 보조 Y2가 비교적 안정적으로 연결된다
- 확장 working data : 2019 ~ 2024
	- 2019년도 데이터는 메인 Y가 분리 문항이라 별도 합성 규칙이 필요하고, 보조 Y과 보조 Y2는 직접 대응이 약하므로 참고용으로만 사용한다
## 공통 원칙
- raw 파일은 수정하지 않는다
- working 폴더에 subset -> rename -> cleaned -> appended 파일을 순서대로 저장한다
- 문항 번호가 아니라 문항 의미 기준으로 변수를 연결한다
- 공백, 비해당, 무응답은 일단 N/A 로 처리하고 코드북 확인 후 확정한다
- 0은 실제 응답값이므로 결측으로 간주하지 않는다 
- 연도별 가중치 변수는 weight로 통일한다
- 전처리 로그(log.md)에는 무엇을 바꿨는지와 왜 그렇게 바꿨는지를 함께 기록한다
	- 전처리 로그표 형식 : 날짜 / 파일 / 변수명 / 처리단계 / 처리 전 상태 / 처리 내용 / 처리 이유 / 처리 후 상태 / 관측치 변화 (N) / 비고
